## Intelligence Layer — THE REAL GAP (Fixed)

### Problem
The 3/10 critic review: "Self-optimization doesn't optimize. There is no intelligence being tiled."
The core pipeline had no actual model invocation. Deadband detected problems but nothing acted on them.

### Solution: model_gate.py + pipeline.py + benchmark.py
- **model_gate.py** — α dial that actually decides code vs model. α=0: never model. α=1: always model. In between: invoke based on code confidence.
- **pipeline.py** — Multi-room signal chain with early exit. When a room resolves with high confidence, downstream rooms are skipped.
- **benchmark.py** — Comparison framework (signal chain vs uniform model)
- **mock_backend.py** — Simulates model responses for testing

### Key Pipeline Feature: Early Exit
```python
# When code resolves confidently, skip downstream rooms
if tile.confidence >= self._early_exit_threshold and tile.label in self._exit_labels:
    result.early_exit_room = room.name
    break  # Stop processing downstream rooms
```
